# 📧 PHP Gmail Integration Website

This project is a simple web application built using HTML, CSS, and PHP that allows users to send emails through Gmail using SMTP authentication.

## 🚀 Features
- User-friendly contact form
- Gmail SMTP integration using PHP
- Secure email sending with authentication
- Responsive UI design
- Clean and minimal layout

## 🛠️ Technologies Used
- HTML5
- CSS3
- PHP
- Gmail SMTP (via PHPMailer)

## 📂 How It Works
1. User fills out the contact form
2. PHP script processes the data
3. Email is sent using Gmail SMTP server
4. Confirmation message is displayed

## ⚙️ Setup Instructions
1. Clone the repository
2. Add your Gmail credentials in the PHP file
3. Enable "App Passwords" in your Google account
4. Run the project on a local server (XAMPP / WAMP)

## ⚠️ Note
For security reasons, never upload your real email password. Use environment variables or app passwords.
